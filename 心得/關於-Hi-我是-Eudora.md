## Eudora
出社會第一份正式工作到職前，決定為自己選一個有意義的名字。

「Eudora」 — — 源自希臘語，有「美好的禮物」之意。當時懷抱著對新工作的期待，期許自己加入公司團隊能猶如一份美好的禮物，帶進好的影響力於工作產出及團隊氛圍。

雖然而後的故事情節不如預期，甚至一度對這個名字感到反感、懷疑自己的價值，但回頭再看，也是一個有所收穫的過程。

如今轉換了環境，重新歸零自己，設定了新的目標並付諸行動於每一日的積累，重新經營自己的名。深知此刻的能量還遠遠不夠，但那些具有影響力的人，都是從 0 開始成長的吧！

開始一段重建的旅程，重拾「Eudora」的初心。

## Blog 文章分類

- **程式之路**
記錄著程式相關的知識、個人筆記、學習歷程。－Ruby On Rails / HTML / CSS / JavaScript …等。
（若內容有誤敬請留言指正，大寫的感謝！）

- **數位工具**
自知自己是個重度依賴數位工具的人，在這裡推薦個人常用的軟體產品與應用方式分享。

- **設計/攝影**
非專業級的攝影玩家，但認為攝影是一種日常的美感練習。不論設備等級，只需一雙能發現「美」的眼睛。
也分享設計相關的知識、作品、軟體操作應用。

- **職場觀察**
分享在職場生活中所領域的小道理、潛規則。

- **悟書/觀影/聽講**
閱讀心得、電影觀後心得、講座參與心得。

- **隨筆隨記**
較隨心、隨性的文章，主要記錄著進修及工作相關的生活瑣碎感想。
